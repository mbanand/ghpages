shiny.webawesome announcement prototype

Note

This is a working prototype for the eventual r-posts.com / R-bloggers announcement. The first goal is to prove that Quarto, Shinylive, webR, R-universe, and the shiny.webawesome runtime can all work together.

TL;DR

shiny.webawesome brings Web Awesome components to R Shiny through generated R wrappers, Shiny bindings, and a bundled Web Awesome runtime.

server <- function(input, output, session) {
  output$message <- renderText({
    count <- input$try %||% 0
    if (count == 0) {
      "Waiting for a click."
    } else if (count == 1) {
      "Shiny received the Web Awesome button action."
    } else {
      paste("Shiny has received", count, "button actions.")
    }
  })
}

shinyApp(ui, server)
